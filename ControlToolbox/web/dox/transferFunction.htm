Transfer Functions

Ensure the propert transfer function is selected to ensure convergence to a solution.
When setting up a network, in addition to selecting the Transfer Function there may be tunable settings for it. See the list below for available options.
Gaussian f(x) =(e^*-(x^2) / (2 * sigma^2)) The graph of a gaussian is a characteristic symmetric "bell curve" shape that quickly falls off towards zero.
The parameter a is the height of the curve's peak, b is the position of the center of the peak, and c (the standard deviation) controls the width of the "bell".
Gaussian functions are widely used in statistics where they describe the normal distributions, in signal processing where they serve to define Gaussian filters, in image processing where two-dimensional Gaussians are used for Gaussian blurs, and in mathematics where they are used to solve heat equations and diffusion equations and to define the Weierstrass transform.
  • transferFunction.sigma - width of the bell?

Linear f(x) = slope*x
In calculus, analytic geometry and related areas, a linear function is a polynomial of degree one or less, including the zero polynomial (the latter not being considered to have degree zero).
Log f(x) = log(x)
  • transferFunction.slope - Slope of the line

Ramp The ramp function generates a straight line with the given slope for all inputs within the acceptable range. Inputs outside of the range get clamped to the high or low limit.
  • transferFunction.slope - Slope of the line
  • transferFunction.xHigh - the high input limit
  • transferFunction.xLow - the low input limit
  • transferFunction.yHigh - the high output limit
  • transferFunction.yLow - the low output limit

Sgn The Sgn function returns a 1 for anything > 0 and a -1 for anything < 0
Sigmoid f(x) = 1/(1+ e^(-slope*x))
A sigmoid function is a mathematical function having an "S" shape (sigmoid curve). It is used in modeling systems that saturate at large values of the argument, or another example is the ogee curve as used in the spillway of some dams. A wide variety of sigmoid functions have been used as the activation function of artificial neurons, including the logistic and hyperbolic tangent functions. Sigmoid curves are also common in statistics as cumulative distribution functions, such as the integrals of the logistic distribution, the normal distribution, and Student's t probability density functions.
  • transferFunction.slope

Sin f(x) = sine(x) The Sin function returns the Sine value of the input.
Step The step function returns the high output limit for x > 0 and the low output limit for x <= 0
  • transferFunction.yHigh - the high output limit
  • transferFunction.yLow - the low output limit

Tanh The Tanh function returns 1 for any input > 100 and -1 for any input > -100. For values within this range the following formula is used: f(x) = (e^(slope*x)-1)/(e^(slope*x)+1)
Settings:
  • transferFunction.slope

Trapezoid The Trapezoid function:
For any x that is both >= Left and Right high: f(x) = 1
For any x that is > leftLow and < leftHigh: f(x) = (x - leftLow)/(leftHigh - leftLow)
For any x that is > rightHigh and < right Low: f(x) = (rightLow -x)/(rightLow - rightHigh)
For any other x : f(x) = 0
Settings:
  • transferFunction.leftLow
  • transferFunction.leftHigh
  • transferFunction.rightLow
  • transferFunction.rightHigh
