Control Point

Control points can be Inputs, Outputs or Set Points. All point types interact with a Mango point in some way. Inputs are used to drive the Controller be recieving data from the real world. Read type data points are useful for Control Input points. Outputs are used to return the control output to the real world. Write type data points are required for Control Output points. Set Points are used to All data points have the following settings:
  • Enabled - if the point is used in the controller
  • Point Type - see above
  • Delay - how many sample periods should the values for this point be offset to the others
  • High Limit - highest possible value (for outputs)
  • Low Limit - lowest possible value (for outputs)
  • Data Point - the Mango data point that is used
  • Controller - the controller that uses this point
