A data point will record one measurement from the station in a standard unit. To change units simply assign the unit below to the data point's unit property, then choose a custom display unit and finally set the text renderer to use the unit as the suffix.

Available Measurements

  • Inside temperature: ℃
  • Outside temperature: ℃
  • Wind speed: km/h
  • Wind direction: degrees from true north
  • rain: .01 inch increments (this is a running total)
  • humidity: %
  • pressure: hPa
